# Changelog

## 1.2.0 (2026-03-21)

### Features

* **lab03:** агентная модель Daisyworld ([lab03](labs/lab03/))
  - Реализация модели Daisyworld на Agents.jl с двумя видами маргариток (чёрные и белые)
  - Механизм температурной саморегуляции через альбедо поверхности
  - Два сценария изменения солнечной светимости: `default` и `ramp`
  - Параметрическое исследование: `init_white` (0.2, 0.8) и `max_age` (25, 40)
  - Визуализация: тепловые карты, анимации, временные ряды, трёхпанельные графики
  - Литературное программирование через Literate.jl (скрипт, notebook, markdown)
  - Отчёт и презентация в формате Quarto
  - Проект организован с помощью DrWatson для воспроизводимости
